# Mortar VS Code Extension

Mortar adds editor visibility for generated SQL in Java projects that use
Mortar metadata and SQL snapshot files.

The extension contributes commands for copying generated SQL and running a
PostgreSQL `EXPLAIN` through the Mortar CLI. It starts `mortar-lsp` from the
configured path and expects project metadata generated by the Java annotation
processor.

This package is pre-release. Packaging a VSIX validates the installable
extension shape; it does not publish the extension to the VS Code Marketplace.
